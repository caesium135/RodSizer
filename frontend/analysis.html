Analysis Instructions

1. Image Review: center panel shows the processed particle image by default. Use the toggle button to switch to the binary mask view if needed.

2. Verify & Exclude: Check the Ledger (right panel). If a detection is incorrect (e.g., scale bar, dirt, not a rod), uncheck the 'nanoparticles' box for that ID to exclude it.

3. Update: Press "Update Particles" to generate statistics and histogram plots based on valid selections.

4. Calibrate if needed: If embedded scale data is unavailable, use the manual scale tool to set the image calibration.

5. Save & Export:

  • Download Excel: Get the raw data for this specific image.
  • Save for Folder Analysis: Confirms these results for the aggregate folder report.
