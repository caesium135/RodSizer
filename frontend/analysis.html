Analysis Instructions

1. Image Review: center panel shows two views:

  • Left: Segmented Nanoparticles (Raw detection).
  • Right: Fitted AuNRs using Rotated Bounding Boxes (Final sizing).

2. Verify & Exclude: Check the Ledger (right panel). If a detection is incorrect (e.g., scale bar, dirt, not a rod), uncheck the 'nanoparticles' box for that ID to exclude it.

3. Calculate: Press "Calculate Analysis" to generate statistics and histogram plots based on valid selections.

4. Save & Export:

  • Download Excel: Get the raw data for this specific image.
  • Save for Folder Analysis: Confirms these results for the aggregate folder report.
