Nanorod Detector 2.0

Detects, sizes, and analyzes gold nanorods.

About This App

This application automates the detection and analysis of gold nanorods from TEM images.

Methodology: Based on JACS Au 2021, 1, 316−327.

Technique: It utilizes K-means clustering for segmentation and fits particles using Rotated Bounding Boxes to extract precise Length and Width measurements.

Development & Enhancements

Improved by Arda & Antigravity, turning the original academic method into a fully functional web application. Key enhancements include:
  • Interactive Dashboard & Folder System
  • Automated Graphs & Histograms
  • Precise Data Export & Reports

Upload Instructions

You can upload multiple images at once.

Supported image formats: TIFF (.tif/.tiff), PNG, JPG/JPEG

Supported metadata formats: .dm3, .dm4, .emd

Scale bar calibration:

  • .dm3/.dm4/.emd: Pixel size read automatically from metadata
  • TIFF (ImageJ): Pixel size read from ImageJ metadata if available
  • PNG/JPG (or TIFF without metadata): Manual calibration is available in the analysis page

If you have a separate metadata file (.dm3/.dm4/.emd), upload it with the same filename as the image for auto-linking.

Note: Please open and review each image individually to verify detection before viewing the Folder Analysis.
